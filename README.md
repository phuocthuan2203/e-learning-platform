# E-Learning Platform

A modern e-learning platform for online courses using Angular, ASP.NET Core, and MySQL.

## Features
- User registration & login
- Profile management
- Course catalog (future)

## Getting Started
1. Clone the repo
2. See frontend/backend setup instructions below

## Technologies
- Angular 20.3.5
- ASP.NET Core 8.0
- MySQL 8.4.6
